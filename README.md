# Torse Studio - React + Tailwind + GSAP landing page

Torse Studio is a website for fictional interior design studio. It was built mostly for practicing animations with GSAP.

## Features

- Fully responsive - optional viewing experience on mobiles, tablets and desktops
- Light / Dark theme - theme is selected based on browser preferences

## Install

Windows

```
git clone https://github.com/adammKass/torse.git

```
